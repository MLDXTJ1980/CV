Bewerbung als Software Quality Assurance in Berlin ab sofort

Sehr geehrte Damen und Herren,

als IHK geprüfte Fachinformatikerin(2020) bewerbe ich mich bei Ihnen auf die Stelle "Software Quality Assurance". Während meines Praktikums habe ich als Quality Assurance mit dem Schwerpunkt "Automatisiertes Frontend Testing" gearbeitet. Anschließend war ich beruflich als Software Testerin bei der G&H Bankensoftware AG tätig.

Ich habe Projekt-Erfahrungen mit Selenium automatisierten Test-Frameworks wie WebdriverIO und Webdriver, und darüber hinaus Kenntnisse in REST und SOAP API Testing Tools( Postman, SoapUI, JMeter ). Außerdem sind mir RESTful APIs automatisierte Testing-Frameworks Rest-Assured Java und Karate bekannt. Daneben runden Berufserfahrungen mit den Schwerpunkten Software Test Analyse und Umsetzung sowie das Funktionale Testing von Bankensoftware mein Profil ab.

Ich verfüge über folgende Kenntnisse:

  • JavaScript, HTML5/CSS3, PHP, Java, C++
  • jQuery, Bootstrap und zusätzliche Libraries für Frontend Entwicklung
  • Responsive Web Design, CSS Flexbox, CSS Grid, Less
  • Erfahrungen mit automatisierte Frontend-Testing Frameworks Selenium WebDriverIO/WebDriverIO
  • Erfahrungen mit automatisierte API-Testing Frameworks REST Assured / Karate
  • Erfahrungen mit MySQL und managementsystem Squirrel SQL
  • vertraut mit Software Testing Tools , z.B Postman, SoapUI, Squirrel
  • Server Side Technologie, RESTFull Services, AJAX, JSON and XML sind mir auch gut bekannt
  • Grundkenntnisse in Git, GitLab CI/CD
  • Grundkenntnisse in Docker
  • PhotoShop und Büro-Anwendungen
  • Grundkenntnisse über Linux und VMware

Neben den fachlichen Kenntnissen verfüge ich zusätzlich über ein TELC B2 Zertifikat für die deutsche Sprache und Kommunikationsfähigkeiten.

Eine hohe Lernbereitschaft, großes Engagement für die Aufgabe, Zuverlässigkeit und Teamfähigkeit runden mein Profil ab. Gerne erzähle ich Ihnen mehr in einem persönlichen Gespräch.

Mit freundlichen Grüßen

Shanshan Zhao
